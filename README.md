## CD-Lab-Work
This program splits a sentence into tokens and classifies each as a keyword, identifier, literal, operator, or separator.

## Output

```text
this  -> Identifier
is    -> Keyword
a     -> Identifier
int   -> Keyword
sum   -> Identifier
=     -> Operator
10    -> Literal
;     -> Separator
and   -> Keyword
a     -> Identifier
+     -> Operator
b     -> Identifier
=     -> Operator
20    -> Literal
```
